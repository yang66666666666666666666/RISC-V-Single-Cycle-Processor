### 关键路径时序
```
关键路径延迟分析（加载指令 lw）:

PC稳定 ──┬── 2ns ──┬── 指令内存输出稳定
         │         │
         └─────────┬── 1ns ──┬── 控制信号稳定
                   │         │
                   └─────────┬── 2ns ──┬── 寄存器读出稳定
                             │         │
                             └─────────┬── 3ns ──┬── ALU输出稳定
                                       │         │
                                       └─────────┬── 2ns ──┬── 数据内存输出稳定
                                                 │         │
                                                 └─────────┬── 1ns ──┬── 结果选择稳定
                                                           │         │
                                                           └─────────┬── 1ns ──┬── 寄存器写入建立

总延迟: 2 + 1 + 2 + 3 + 2 + 1 + 1 = 12ns
最大频率: 1/12ns ≈ 83.3MHz
```
