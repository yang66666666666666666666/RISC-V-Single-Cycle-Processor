## 🔧 快速解码方法

### 通过opcode识别指令类型:
- `0010011` (0x13) → I型算术指令 (addi, slli, slti等)
- `0110011` (0x33) → R型算术指令 (add, sub, and, or等)
- `0110111` (0x37) → U型指令 (lui)
- `0010111` (0x17) → U型指令 (auipc)
- `0100011` (0x23) → S型指令 (sw, sh, sb)
- `0000011` (0x03) → I型加载指令 (lw, lh, lb等)
- `1100011` (0x63) → B型分支指令 (beq, bne等)
- `1101111` (0x6F) → J型跳转指令 (jal)
- `1100111` (0x67) → I型跳转指令 (jalr)

### 通过funct3识别具体操作:
**I型/R型算术指令:**
- `000` → ADD/ADDI 或 SUB (看funct7[5])
- `001` → SLL/SLLI (左移)
- `010` → SLT/SLTI (有符号比较)
- `011` → SLTU/SLTIU (无符号比较)
- `100` → XOR/XORI (异或)
- `101` → SRL/SRLI 或 SRA/SRAI (右移，看funct7[5])
- `110` → OR/ORI (或)
- `111` → AND/ANDI (与)

这个指令序列完美展示了RISC-V处理器的核心功能，包括所有主要的指令类型和操作！
