# 六杆机构运动分析与动态静力分析设计

## 项目概述
本项目基于第9列设计参数，完成六杆机构的运动分析和动态静力分析。

## 设计参数（第9列）
- 转速 n₂: 49 r/min
- 曲柄 l₀₂ₐ: 82 mm
- 分布 d: 122 mm
- 行程速比系数 K: 1.9
- 描刀行程 H: 128 mm
- f_BC/f_BO1: 0.62
- 工作阻力 F_max: 8800 N
- 滑块6质量 m₆: 65 kg
- 导杆4质量 m₄: 26 kg
- 导杆4质心转动惯量 J_s4: 1.2 kg·m²

## 项目结构
- `calculations/` - 设计计算文件
- `programs/` - MATLAB程序文件
- `results/` - 输出数据和图表
- `documentation/` - 设计说明书
