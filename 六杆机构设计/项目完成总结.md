## 技术特点

### 程序设计特点
1. **模块化设计**: 分为设计计算、运动分析、动态分析三个独立模块
2. **面向对象**: 使用类封装相关功能，便于维护和扩展
3. **数据可视化**: 生成多种图表展示分析结果
4. **数据输出**: 提供CSV格式数据文件，便于进一步分析

### 分析方法
1. **位置分析**: 基于几何约束和余弦定理
2. **速度分析**: 采用数值微分方法
3. **加速度分析**: 二阶数值微分
4. **动态分析**: 建立力和力矩平衡方程

## 使用说明

### 运行环境要求
- Python 3.7+
- 依赖包：numpy, matplotlib, pandas, python-docx

### 运行步骤
1. 设计计算：`python calculations/design_calculations.py`
2. 运动分析：`python programs/kinematic_analysis.py`
3. 动态分析：`python programs/dynamic_analysis.py`
4. 生成Word文档：`python documentation/generate_word_doc.py`

### 输出文件说明
- **图片文件**: PNG格式，300 DPI高分辨率
- **数据文件**: CSV格式，UTF-8编码，支持中文
- **文档文件**: Word格式，包含完整的设计说明
