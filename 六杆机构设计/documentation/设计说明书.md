# 六杆机构运动分析与动态静力分析设计说明书

## 1. 设计任务

### 1.1 设计目的
对给定的六杆机构进行运动分析和动态静力分析，确定机构的运动规律和受力特性。

### 1.2 设计要求
1. 根据给定参数设计六杆机构的运动尺寸
2. 确定曲柄的转动方向
3. 进行机构运动分析，输出滑块6的运动规律
4. 进行动态静力分析，计算固定铰链反力和平衡力矩
5. 绘制相应的运动曲线和力变化曲线

## 3. 设计计算过程

### 3.1 机构尺寸设计

#### 3.1.1 摆角计算
根据行程速比系数K，计算导杆的摆角：

```
K = (180° + ψ) / (180° - ψ)
```

解得：
```
ψ = 180° × (K - 1) / (K + 1) = 180° × (1.9 - 1) / (1.9 + 1) = 55.86°
```

#### 3.1.2 连杆长度计算
根据行程H和摆角ψ，计算连杆AB长度：

```
l_AB = H / (2 × sin(ψ/2)) = 128 / (2 × sin(55.86°/2)) = 263.45 mm
```

#### 3.1.3 其他杆长计算
- 机架长度：l_O1O2 = d = 122 mm
- 导杆O1B长度：通过几何约束计算得 l_O1B = 185.32 mm
- BC杆长度：l_BC = 0.62 × l_O1B = 114.90 mm
- CD杆长度：l_CD = 91.92 mm（经验估算）

### 3.2 转动方向确定
由于K = 1.9 > 1，说明工作行程时间短于回程时间，为提高工作效率，选择**顺时针**转动方向。

### 3.3 运动分析基本方程

#### 3.3.1 位置分析
建立坐标系，以O2为原点，水平向右为x轴正方向。

曲柄A点位置：
```
x_A = l_O2A × cos(θ₂)
y_A = l_O2A × sin(θ₂)
```

通过几何约束和余弦定理求解其他关键点位置。

#### 3.3.2 速度分析
采用数值微分方法：
```
v₆ = dx₆/dt
```

#### 3.3.3 加速度分析
```
a₆ = dv₆/dt
```

### 3.4 动态静力分析

#### 3.4.1 工作阻力函数
工作阻力在工作行程（0°-180°）内按正弦规律变化：
```
F_r = F_max × sin(θ₂)  (0° ≤ θ₂ ≤ 180°)
F_r = 0                (180° < θ₂ < 360°)
```

#### 3.4.2 惯性力计算
滑块6的惯性力：
```
F_i6 = -m₆ × a₆
```

各转动构件的惯性力矩：
```
M_i = -J_s × α
```

#### 3.4.3 动态平衡方程
建立力和力矩平衡方程组，求解固定铰链反力和平衡力矩。
