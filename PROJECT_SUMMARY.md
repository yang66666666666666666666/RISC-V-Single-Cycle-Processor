# RISC-V Single Cycle Processor - Project Summary

## 🎯 项目概述

本项目实现了一个完整的RISC-V单周期处理器，专门为Windows环境优化，支持多种仿真和综合工具。

## ✅ 已完成的功能

### 核心处理器设计
- ✅ 32位RISC-V架构
- ✅ 单周期执行模型
- ✅ 完整的数据路径设计
- ✅ 控制单元实现
- ✅ ALU支持多种运算
- ✅ 寄存器文件（32个寄存器）
- ✅ 指令和数据内存

### 支持的指令集
- ✅ **I-type**: `addi` (立即数加法)
- ✅ **R-type**: `add`, `sub`, `and`, `or`
- ✅ **Load/Store**: `lw`, `sw`
- ✅ **Branch**: `beq` (基础支持)

### 开发环境支持
- ✅ **Vivado项目文件** (Windows主要支持)
- ✅ **Icarus Verilog支持** (跨平台)
- ✅ **自动化脚本** (Windows .bat文件)
- ✅ **Makefile** (Linux/Unix支持)

### 测试和验证
- ✅ **完整的测试程序**
- ✅ **详细的仿真输出**
- ✅ **波形文件生成**
- ✅ **自动化测试脚本**

## 🚀 使用方法

### Windows用户（推荐）
1. **一键运行**: 双击 `demo.bat` 选择运行方式
2. **Vivado**: 运行 `run_vivado.bat` 创建项目
3. **手动**: 按照 `WINDOWS_SETUP.md` 操作

### Linux/Unix用户
```bash
make iverilog  # 编译并运行仿真
```
